# Calculadora Hipotecaria — Cataluña

Calculadora de préstamo hipotecario con todos los gastos reales de compraventa, específica para **Cataluña**.

## Características

- **Sistema de amortización francés** (cuota constante)
- **Tipo de vivienda:** Segunda mano (ITP 10%) u Obra nueva (IVA 10% + AJD 1,5%)
- **Impuestos de Cataluña** calculados automáticamente y editables
- **Desglose completo de gastos:** Notaría, Registro de la Propiedad, Gestoría, Tasación, Comisión de apertura, Seguros
- **Liquidez total necesaria** el día de la firma
- **Coste total de la operación** (hipoteca + intereses + gastos)
- **Amortizaciones anticipadas** con cálculo del ahorro
- **Tabla de amortización** mensual paginada
- **Gráfico interactivo** de evolución capital / intereses / capital pendiente

## Tecnologías

- HTML5 + JavaScript (vanilla)
- Tailwind CSS
- ApexCharts
- Font Awesome

## Uso

Abre `index.html` directamente en el navegador. No requiere servidor ni dependencias adicionales.
